HTML / CSS

Various classes

You will see a lot of divider-* classes, they work everywhere and on any element that you want to verticaly separate from another element or make vertical spacing between two elements.

e.g. divide-xs have top and bottom padding of 7px, -sm 30px, -md 45px, -lg 60px and -xl 90px.

<div class="divide-sm">...</div>

You also have standard HTML tag <hr> and maybe you will want to remove top margin, etc. We were preapred for that. Just add class to <hr> element:

e.g. no-margin will remove all margins, no-margin-top will remove top margin, no-margin-bottom will remove bottom margin.

<hr class="no-margin-top">

For <hr> element you also have class title-divider for undertitles:

<hr class="title-divider">

Colors

Add white-bg or smoke-bg as class on element for background color on element:

<section class="white-bg">...</section>
<section class="smoke-bg">...</section>

For text we prepared classes light and smoke:

<h5 class="smoke">Subtitle</h5>
<h1 class="light">Main title</h1>

The fastest way to change all blue things on page is to open style.css and with ‘replace all’ command in your editor find all #00a0dc, which is default blue color. There is also one #008cc8, which is darker blue button hover state color.

With this you have changed buttons, icons and input field labels which were our default blue.

You don’t know what color to use? See: http://www.materialui.co/colors

Need some famous brand colors? See: http://brandcolors.net/

Fonts / Typography

We have only used Google fonts in our demo, you can use any with in a minute.

Default is: Open Sans for paragraph, Montserrat for titles and Droid Serif for lower titles.

In index.html inside head replace our link to font with your desired one.

<link href='http://fonts.googleapis.com/css?family=Open+Sans:300,400,700' rel='stylesheet' type='text/css'>
<link href='http://fonts.googleapis.com/css?family=Montserrat:400,700' rel='stylesheet' type='text/css'>
<link href='http://fonts.googleapis.com/css?family=Droid+Serif:400,400italic' rel='stylesheet' type='text/css'>

Under 2) Typography in style.css find font-family: property and change it.

h1, h2, h3, h4 {
    font-family: 'Montserrat', sans-serif;
    font-weight: 400;
    margin: 0 0 15px 0;
}

h5 {
    font-family: 'Droid Serif', serif;
    font-size: 19px;
    font-style: italic;
    line-height: 1.48;
    margin: 0 0 15px 0;
}

SVG icons

In different sections icons have different sizes and properties, we made:

myway-icon-services, myway-icon-features, myway-icon-gallery, myway-icon-team and myway-icon-info

You can edit there size and color in style.css

<div class="myway-icon-team">
    ...
</div>

Next step is choosing the icon. We have included two sets of icons (plus social) and you can find them in CSS folder. All you need to do is to open svg file in your text editor and copy #ID of icon into code.

<use xlink:href="#facebook"></use>

A lot of other icons can be found on www.iconmelon.com. Select icons that you would like to use, download SVG file and move it into CSS folder. Open custom.js file and under 6) SVG icons add link to your new SVG file.

Iconmelon

Final icon look like this:

<div class="myway-icon-team">
    <svg viewBox="0 0 32 32">
        <use xlink:href="#facebook"></use>
    </svg>
</div>

Preloader

Right after <body> tag you can find div with preloader class.

Go to style.css file and at the end of file find 14) Preloader and .spinner > div where you can change default background-color: #aaa; property which stands for those three dots color and .preloader where you can change background-color: #fff; which stands for white preloader background.

If you would like to remove preloader:

  • Remove preloader part of code from HTML
  • Remove all CSS under 14) Preloader in style.css
  • Remove 11) Preloader part of code in custom.js

Header overlay

In style.css1 under 5) Intro & Sections Headers you can find .section-overlay with background: rgba(0, 0, 0, .5); property.

Change 0, 0, 0 as RGB color code. 0, 0, 0 is black and 255, 255, 255 is white.

Whant blue overlay? e.g:

.section-overlay {
    background: rgba(0, 160, 220, .5);
    position: absolute;
    width: 100%;
    height: 100%;
}

Change .5 for transparency where 0 is unvisible, 1 is visible and .5 is 50% transparent.

Whant less transparency? e.g:

.section-overlay {
    background: rgba(0, 160, 220, .8);
    position: absolute;
    width: 100%;
    height: 100%;
}

Scripts

Carousel

Every carousel MUST have it’s own #ID! After creating carousel you should add it’s #ID in to custom.js file under 3) Carousels.

e.g.

$('#carousel-1, #carousel-2, #carousel-3').carousel({
    interval: false
})

Setup carousel:

  • interval: 4000 - Change this for duration between slides in miliseconds. Leave 0 for no auto sliding.
  • pause: "false" - Stands for pausing slider on mouse hover.

Every carousels left and right buttons MUST point to it’s carousel ID with e.g. href="#carousel-1"

<button class="left carousel-control" href="#carousel-1" data-slide="prev"><i class="fa fa-angle-left"></i></button>
<button class="right carousel-control" href="#carousel-1" data-slide="next"><i class="fa fa-angle-right"></i></button>

Retina support

When device with retina display open a page, script checks each image on the page to see if there is a high-resolution version of that image on your server. If a high-resolution variant exists, the script will swap in that image in-place.

For example, if you have an image on your page like this "/img/my_image.png" script will check your server to see if an alternative image exists at this path "/img/my_image@2x.png"

Simplified, all you have to do is to include same image with double width and height and @2x added in file name, and in same folder as original image on your server. If you don’t include double sized image then script will leave original one.

For image that you want to check for double sized one everything is normal in HTML, e.g.:

<img src="img/gallery/gallery-01.jpg" alt="">

If you dont want script to search for retina image add data-no-retina attribute on img element. e.g.:

<img src="img/gallery/gallery-01.jpg" data-no-retina alt="">

Google Maps

  • center: - Change latitude and longitude for map center
  • zoom: - Adjust zoom level
  • position: - Change latitude and longitude for marker

Pin image can be found in IMG folder.

For map styling go to Snazzymaps website and pick your desired map styling. Copy JavaScript Style Array in custom.js file under style: property of 7) Google map.
